Nicola découvre un compas dans la trousse à dessin que lui a offert Sofia. Il décide d'apprendre à s'en servir.

Par trois points qui ne sont pas alignés (c'est à dire que l'on ne peut pas trouver de droite sur laquelle ces points se trouvent tous les trois), il passe un seul et unique cercle. Les trois points sont représentés par une chaîne de caractère qui contient leurs coordonnées, de la façon suivante : 

    "(x1,y1),(x2,y2),(x3,y3)"

où x1,y1,x2,y2,x3,y3 sont des nombres.

Il vous faut déterminer le cercle qui passe par ces trois points, et retourner le résultat sous forme d'une chaîne de caractères qui présente l'équation du cercle. Dans une repère cartésien (d'axes X et Y), le cercle dont le centre a pour coordonnées (x0,y0) et le rayon est r peut être décrit par l'équation suivante : 

    "(x-x0)^2+(y-y0)^2=r^2"

où x0,y0,r sont des nombres décimaux arroundis à deux décimales. Pour arrondir, utiliser les règles mathématiques standard. Enlever les zéros surnuméraires.

Input: Les couples de coordonnées (chaîne de caractères - str).

Output: L'équation du cercle (str).

Exemple: 

checkio("(2,2),(6,2),(2,6)") == "(x-4)^2+(y-4)^2=2.83^2"
checkio("(3,7),(6,9),(9,7)") == "(x-6)^2+(y-5.75)^2=3.25^2"

Utilisation: Cette équation, connue sous le nom d'"équation du cercle", exprime que le carré de la distance euclidienne du centre de coordonnées (x0, y0) au point courant du cercle de coordonnées (x, y), qui est (x0 - x)**2 + (y0 - y)**2, égale le carré du rayon du cercle r**2. Le cercle est un très bel objet mathématique !

Préconditions: Les trois points donnés ne sont pas alignés.
0 < xi, yi, r < 10
